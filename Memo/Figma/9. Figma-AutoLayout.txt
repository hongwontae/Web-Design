1. Auto Layout
1-1. Frame에 적용되는 속성이며 그 안에 요소들을 자동으로 정렬하고 크기를 동적으로 조절할 수 있게 해줍니다.
1-2. Frame에서 패딩을 설정하면 프레임 테두리에서부터 지정된 값까지 4분면에서 크기를 차지합니다.
=> 이 떄 남은 공간을 Auto Layout 하위 속성인 Alignment를 통해 정렬할 수 있고 이 공간을 정렬 가능한 공간이라고 합니다.
1-3. Auto Layout 탭을 통해 Padding을 설정하거나 Alignment를 설정할 수 있습니다.
1-4. Auto Layout를 적용하면 default는 가로 정렬입니다. (flex의 row와 비슷합니다.)
1-5. 프레임 크기가 작고 패딩크기가 크다면 -> 내부 요소가 잘립니다.
1-6. 프레임을 먼저 auto layout 설정 후 요소를 집어넣으면 자동으로 요소가 layout의 영향을 받습니다.


2. 여러 요소 Auto Layout
2-1. 프레임 내에 여러 요소가 존재하고 이를 단체로 정렬하고 싶을 떄 사용합니다.
2-2. 여러 요소를 선택하고 auto layout를 설정하거나 프레임 내부에 여러 요소를 생성하고 auto layout를 설정합니다.
2-3. 요소들은 자동으로 수평 맞춤과 갭 맞춤이 됩니다.
2-4. padding과 gap, alignment 사용 가능합니다.

* 1,2에서 프레임을 늘리거나 줄이면 자동으로 요소들이 적절하게 배치됩니다.


3. Text 기준선 정렬
3-1. Auto Layout Settings에서 Align text baseline을 통해 조절할 수 있습니다.


4. Auto Layout의 영향을 벗어나는 요소 만들기
4-1. Auto Layout를 설정한 프레임에서 배치할 요소를 넣습니다.
4-2. 요소는 레이아웃 흐름에 따른 배치를 따릅니다.
4-3. 이 떄 해당 요소를 선택하고 속성 패널의 position에서 
 Ignore auto layout를 클릭하면 흐름을 벗어나 자유롭게 배치할 시킬 수 있습니다.


5. Auto Layout과 반응형
5-1. Fixed Widht/Height
    1. 항상 고정된 크기를 갖습니다.
    2. 패딩을 늘리거나 줄이면 고정된 프레임 크기 내에서 공간을 차지합니다.
    3. 프레임을 줄이거나 크게할 수 있습니다.
5-2. Hug Contents
    1. Auto Layout을 적용할 떄 default 값입니다.
    => 요소의 크기에 4분면에 10 padding을 적용한 값
    2. 패딩을 적용하면 외부로 크기와 높이를 차지합니다.
    3. 프레임을 줄이거나 크게하면 fixed로 변경됩니다.
5-3. Fill Container
    1. 부모 요소가 auto layout일 떄 자식 요소의 auto layout에서 사용할 수 있는 옵션입니다.
    2. 부모 요소 프레임에 적용된 패딩 넓이나 높이만큼 자식 요소가 커집니다.
    3. 부모 요소 프레임의 크기에 따라 자식 요소들이 반응형으로 반응합니다.
