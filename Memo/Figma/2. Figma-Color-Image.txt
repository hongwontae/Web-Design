1. Color 조합
1-1. 색상은 무작위로 조합하는게 아닌 정형화된 규칙이 존재합니다.
=> 유사한 색상 조합, 하나의 색상에서 채도 조절, 대비되는 색상 조합 등..
1-2. 그라데이션은 Fill 속성 gradient를 클릭하여 생성하고 조작할 수 있습니다.
1-3. 대체로 피그마에서 색상은 RGB로 제어합니다.

1-4. 색의 성질-조절 방법 용어
    1. 휴 : 색 자체, 순수한 색상을 의미합니다.
    => 빨강, 노랑, 초록 등의 기본 색상 계열입니다. (색상환의 12가지)
    
    2. 셰이드 : 색을 어둡게 만든 것입니다.
    => 명도를 낮춰 무게감 있는 색을 만들고 싶을 떄 사용합니다. (그림자 느낌)

    3. 톤 : 색의 채도를 조절하는 것입니다. (선명함)
    4. 틴트 : 색의 명도를 조절하는 것입니다. (밝기)


2. Image
2-1. 프레임 내부에 도형을 생성하고 이미지를 추가하면 도형 크기에 맞춰 fill 속성으로 채워집니다.
=> fill 속성 : 사진이 포함될 크기에 맞춰 최대한 비율을 유지하여 들어가는 속성입니다. (default)
=> 2-1 방법을 사용하면 사진은 단일 객체가 아닌 도형의 속성값이 됩니다.
2-2. fit 속성
=> 부모 객체 크기를 신경쓰지 않고 자신 원본 비율을 지키며 들어가는 속성입니다.
2-3. crop 속성
=> 사진의 원본 비율을 그대로 보여주고 사용자가 원하는 방식으로 잘라서 들어가는 속성입니다.
2-4. Tile
=> 사진을 n개로 잘라서 부모 크기 안에 나열하는 방법입니다.
