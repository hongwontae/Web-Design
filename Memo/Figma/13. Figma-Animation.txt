1. Animation
1-1. 화면 간 전환이나 요소의 움직임 등을 부드럽게 표현하기 위해 사용하는 동작 효과입니다.
1-2. 프레임 또는 컴포넌트 간의 전환을 설정할 떄 애니메이션 사용 가능합니다.
1-3. Animation은 프레임이나 컴포넌트를 대상으로 작동하지만 스마트 애니메이션은 같은 레이어&&이름 동작합니다.


2. 설정 방법
2-1. Prototype에서 interaction에서 Trigger, Action, Destination를 설정합니다.
2-2. Animation을 설정합니다.


3. 애니메이션 종류
3-1. Instant : 애니메이션 없이 즉시 전환됩니다.
3-2. Dissolve : 점점 흐려지며 사라지고 나타나는 페이드 인/아웃 효과입니다.
3-3. Smart Animation : 같은 이름의 레이어가 있으면 해당 요소의 위치, 크기, 색상 등을 
 자동으로 보간하여 자연스럽게 이동시키는 호과입니다.
(레이어를 선택 후 Interaction으로 smart navigation을 선택하고 프레임을 선택하면 자동으로 일치하는 레이어를 찾아 
 자동으로 애니메이션을 수행합니다.)
3-4. Move in : Destination이 지정된 방향에서 Hotspot으로 들어옵니다.
3-5. Move out : Hotspot이 지정된 방향으로 밀려 나가며 Destination으로 전환됩니다.
3-6. Push : Hostpot이 슬라이드로 밀려나고, Destination이 반대 방향에서 동시에 밀려 들어오는 전환효과
3-7. Slide in/out : Move와 비슷하지만 더 부드러운 효과를 줍니다.


4. 완화 곡선 (Easing Curve)
4-1. 애니메이션 전환 효과의 이동 속도나 선형-비선형을 의미합니다.
4-2. Linear : 선형적으로 효과를 줍니다.
4-3. Ease in : 처음에는 천천히 -> 도달하기 전에 빠르게
4-4. Ease out : 처음에는 빠르게 -> 도달하기 전에 느리게
4-5. Ease in back : 객체가 위로 조금 벗어났다가 처음에는 천천히 -> 도달하기 전에 빠르게
4-6. Ease out back : 처음에는 빠르게 -> 도달하기 전에 객체 아래로 벗어났다가 느리게
4-7. Ease in and out back : 위, 아래 바운스 효고 + 중간 빠르게
