1. Auto Layout, 그룹 취소
1-1. ctrl + shift + g를 통해 취소할 수 있습니다.


2. 프레임 내 요소 위치 고정한 채 프레임 크기 조절
=> ctrl + 드래그


3. 프레임 내에 객체를 직선으로 내리는 방법
=> shift를 누른채 이동하면 직선으로 내려갑니다.


4. 컴포넌트 + 베리언트 공간에서 Prototype를 적용할 수 있습니다.
5. 그리고 컴포넌트를 가져오면 프레임에 위치시켜야 Test를 할 수 있습니다.

6. 프레임 시각 구도 맞추기
=> 프레임을 선택하고 SHIFT + 1


7. interaction 대상, 객체
.. 컴포넌트 variant 까지


8. 스크롤할 떄 객체 고정시키기 (css에서 fix)
=> 객체 선택 -> 프로토타입 -> Scroll behavior -> position -> Fixed 선택


9. 수평 스크롤
** 부모 프레임보다 내부 객체들의 크기가 더 커야 합니다. **
9-1. 객체를 선택하여 프레임의 자식으로 둡니다.
9-2. 프레임의 크기를 줄입니다.
=> 줄일 떄 프레임의 중앙을 맞춰야 합니다. (그래야 양쪽 여백이 동일해집니다.)
9-3. 프레임을 선택 후 prototype에서 Overflow Horizontal를 선택합니다.


10. 객체 선택 후 프레임으로 만들기
=> ctrl + alt + g
