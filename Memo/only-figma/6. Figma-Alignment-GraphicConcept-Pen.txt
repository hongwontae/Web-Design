1. Alignment
1-1. 정렬 옵션은 수평에서 왼-중-오, 수직에서 상-중-하로 정렬시킬 수 있습니다.
1-2. 정렬 옵션은 객체 요소가 두 개 이상일 떄만 적용됩니다.
1-3. Frame 내에서 객체를 클릭 후 Alignment를 사용하면 자동으로 프레임를 기준으로 정렬됩니다,
 => 그러나 Frame이 아닌 도형 객체 내에 도형 객체는 해당되지 않습니다.
 => 두 개의 도형 객체를 선택하고 Alignment를 사용하면 부모 도형 객체 기준으로 정렬됩니다.

1-4. 여러 객체를 동시에 정렬하고 싶다면?
    1. 여러 객체를 수평으로 같은 gap를 가지게 만드는 방법
    => 여러 객체를 선택하고 Alignment의 Distribute horizontal spacing를 사용합니다.

    2. 여러 객체를 수직으로 같은 gap를 가지게 만드는 방법
    => 여러 객체를 선택하고 Alignment의 Distribute Vertical spacing를 사용합니다.

    3. 무작위 객체를 자연스럽게 같은 gap를 가지게 만드는 방법
    => 여러 객체를 선택하고 Alignment의 Tidy up를 사용합니다.

    4. 여러 객체를 프레임이나 부모를 기준으로 동시 정렬하게 만들고 싶다면
    => 여러 객체를 선택하고 alt + Alignmnet 정렬 사용


2. 그래픽
2-1. 레스터 그래픽과 백터 그래픽이 존재합니다.
=> 레스터 그래픽은 픽셀 기반이고 백터 그래픽은 선과 패스입니다.
=> 둘의 큰 차이점은 레스터는 확대/축소할 떄 품질이 떨어지지만 백터는 떨어지지 않습니다.
2-2. 객체를 선택하고 enter를 누르면 Vector 모드로 전환됩니다.
=> 각 모서리의 점들을 직접 수정할 수 있는 백터 객체로 변경됩니다.


3. 펜
3-1. 단축키 p를 눌러서 펜을 생성할 수 있습니다.
=> 첫 번쨰 점을 클릭 후 두 번쨰 점을 더블클릭 하면 일자 선을 만들 수 있습니다.
3-2. 곡선
=> 한 점을 클릭 후 두 번쨰 점을 클릭한 상태에서 마우스를 움직여서 곡선을 만들 수 있습니다.
