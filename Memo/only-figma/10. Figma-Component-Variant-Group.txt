1. Component
1-1. 요소를 컴포넌트화 시키려면 ctrl + alt + k or 마우스 우클릭 후 create component를 클릭하면 됩니다.
1-2. 컴포넌트화 된 요소는 왼쪽 layer 패널에서 보라색으로 보이게 됩니다.
1-3. 컴포넌트를 복사한 객체를 자식 자식 컴포넌라고 합니다. or 메인 컴포넌트의 인스턴스라고도 합니다.
1-4. 왼쪽 패널에서 부모는 4다이아몬드 아이콘을 가지고 자식은 1다이아몬드를 갖습니다.

1-5. 컴포넌트가 되면 왼쪽 패널의 assets에서 가져와 사용할 수 있습니다.
1-6. 자식 컴포넌트는 부모 컴포넌트의 변경에 영향을 받습니다.
1-7. 그러나 자식 컴포넌트를 커스터마이징 했다면 그 컴포넌트는 영향을 받지 않습니다.


2. Variant
2-1. 여러 컴포넌트의 변형을 표현할 수 있는 기능입니다. (컴포넌트화된 요소만 가능)
2-2. 컴포넌트 요소 우클릭 -> Main component -> Add variant or 오른쪽 패널에서 다이아몬드내의 + 도형 클릭
=> variant 생성할 수 있습니다.
2-3. variant를 생성하면 assets에 컴포넌트는 하나이나 가져와서 오른쪽 패널을 통해 자유롭게 변경할 수 있습니다.
2-4. variant의 이름은 Prop=name 형식을 지켜줘야 합니다. (Prop은 키이름 value는 자유롭게)


3. Component 속성
=> 컴포넌트 요소가 되면 4가지 속성을 사용할 수 있습니다.
3-1. Variant
3-2. Boolean
3-3. Instance swap
3-4. Text


4. Group
4-1. ctrl + g를 통해 그룹화할 수 있습니다.
4-2. 레이어 정리, 함께 이동 및 크기 조정, 정렬과 정돈, 빠른 선택과 편집...


4. 단축키
4-1. 인스턴스 컴포넌트에서 부모 컴포넌트로 가는 단축키
=> ctrl + shift + alt + k
4-2. 프레임이나 그룹 내에서 선택한 요소와 동일한 요소를 선택해주는 단축키
=> ctrl + alt + a
