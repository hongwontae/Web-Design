1. Blur
=> 시각적으로 부드러운 흐림 효과를 주는 기능입니다.
=> Layer Blur와 Background Blur 나눌 수 있습니다.

1-1. Layer Blur
=> 선택한 객체를 흐리게 만듭니다.

1-2. background Blur
=> 도형을 생성하고 블러 처리할 공간을 인위적으로 만들어 사용하는 기법입니다.


2. Blur 주의사항
2-1. 프레임 자체에 Blur 처리를 할 수 있으나 프레임 내부의 모든 객체도 Blur의 대상이 됩니다.
2-2. 프레임 내부에 프레임 크기에 맞는 도형을 생성 후 background 처리 후
 도형 자체에 layer blur 처리를 하면 다른 객체들은 영향을 받지 않습니다.


3. Export
3-1. 피그마 프레임이나 객체, 사진을 png, jpg, svg..로 내보낼 수 있습니다.
3-2. 여러 객체를 한꺼번에 보내고 싶다면 그룹으로 만들고 Export하면 됩니다.
