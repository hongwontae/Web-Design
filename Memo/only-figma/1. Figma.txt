1. resize to fit (ctrl + shift + alt + r)
=> 상위 부모가 하위 객체 크기에 맞게 크기를 변환하는 기능입니다.
=> Layout 속성 패널에 아이콘이 있습니다.


2. Grid
2-1. 행 그리드, 열 그리드, 모듈 그리드(행 + 열)로 크게 3가지로 나눌 수 있습니다.
2-2. 격자 그리드, 베이스라인 그리드, 계층적 그리드
2-3. 줄 사이의 공간을 gap, gutter, space라고 부르며, 행과 열이 겹치는 부분을 모듈이라고 합니다.

2-4. grid를 실행하려면 프레임의 속성 패널에서 layout grid를 선택하면 됩니다.
(이 후 shift+g로 끄거나 켤 수 있습니다.)
2-5. 열이나 행을 기준으로 grid를 맞출 수 있습니다.
2-6. 또한 마진이나 gap을 설정할 수 있습니다.
2-7. 설정한 그리드를 여러 프레임에서 사용하고 싶다면
 layout grid -> dot 4개 버튼 클릭 -> Layout guide styles -> + 클릭 -> Create new layout guide style을 통해
 재사용할 수 있는 그리드 스타일을 만들 수 있습니다.
2-8. grid 열은 12개로 나누는 것이 좋습니다. => 다양한 2,3,4로 나눌 수 있습니다. -> 다양한 배치가 가능합니다.
