1. 불리언 연산
1-1. 도형을 조합하거나 잘라내는 기능입니다.
1-2. 연산 종류
    1. Union
    => 두 도형을 합쳐서 하나로 만듭니다.
    => 두 도형 중 앞선 도형의 background를 갖습니다.

    2. Substract
    => 두 도형에 앞선 도형 크기만큼 빼버립니다.
    => 두 도형 중 뒤에 있는 도형의 background를 갖습니다.

    3. Intersect
    => 두 도형에서 겹치는 크기만 갖습니다.
    => 두 도형 중 앞선 도형의 background를 갖습니다.

    4. Exclude
    => 두 도형에서 겹치는 부분만 제외합니다.
    => 두 도형 중 앞선 도형의 background를 갖습니다.
