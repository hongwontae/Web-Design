1. 정렬
=> 정교하고 완성된 web design을 위해서는 정렬은 첫 번쨰 고려사항입니다.
=> 정렬에는 수평축과 수직축 정렬이 존재하고 각 왼쪽, 중앙, 오른쪽 정렬이 존재합니다.


2. 피그마에서 그리드 설정
2-1. Frame을 생성하고 왼쪽 design 탭에서 layout guide를 클릭하면 프레임을 10px 크기의 
 작은 사각형으로 나누는 2차원 그리드를 얻습니다.
2-2. web design에서 가장 널리 사용하는 레이아웃 그리드는 12열 그리드입니다.
=> 6,6/ 4,4,4/ 3,3,3,3... 다양하게 설정할 수 있습니다.
2-3. shift + g를 통해 그리드 표현을 감추거나 나타나게 할 수 있습니다.
2-4. grid tab에서 Column은 n에 따라서 열을 배치하고 row는 n에 따라서 행을 배치합니다.
2-5. grid의 margin은 여백이며 gutter는 열과 열, 행과 행 사이의 공간입니다.


3. Object 다루기
3-1. Object Dimensions에서 w,h를 넓이와 높이를 의미합니다.
=> 넓이나 높이를 클릭 후 위, 아래를 선택하면 1px, shift 위, 아래는 10px 커지거나 작아집니다.
3-2. ctrl c,v로 객체를 복사하거나 alt를 누르고 객체를 드래그하면 객체가 복사됩니다.
3-3. 하나의 객체나 여러 객체를 선택한 후 shift 방향키를 누르면 객체가 프레임 내에서 이동합니다.
3-4. Figma에서 객체를 선택하고 Fill에서 색상을 선택할 수 있습니다. 또한 투명도 조절도 가능합니다.

=> 중요한 내용을 돋보이게 하고 덜 중요한 내용은 눈에 잘 띄어서는 안됩니다.

5. web design에서는 착시현상을 고려해서 디자인을 설계해야 합니다.
=> 같은 크기, 같은 높이더라도 사용자가 보기에는 다르게 느껴진다면 같은 크기나 높이가 아닙니다.
=> 실제로 두 요소가 같은 크기-높이더라도 한 요소가 더 커보인다는 것을 객체의 무게감이라고 합니다.
