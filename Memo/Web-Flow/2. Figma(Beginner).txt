1. Figma
=> WebFlow보다 좁은 의미의 웹 디자인 Tool
=> Drafts를 클릭하여 page를 생성할 수 있습니다.


2. 흐름 및 도구
2-1. 상단의 design을 클릭하면 untitled이라는 파일 생성됩니다.
2-2. 하위 메뉴에 첫 번쨰는 객체를 움직이는 커서
2-3. 두 번쨰는 Frame이며 웹 디자인이 들어갈 화면을 설정할 수 있습니다.
=> space 바를 누르고 드래그하면 화면 어디든 이동할 수 있습니다.
2-4. 세 번쨰는 도형이며 객체입니다.
2-5. 네 번쨰는 텍스트입니다.
2-5. 다섯번쨰는 주석입니다.
2-6. 여섯번쨰는 여러 기능입니다. -> 피그마와 관련된 모든 설정과 옵션
=> ctrl + k 단축키


3. Figma 개념
3-1. 프레임을 생성하면 전체 자식의 부모가 됩니다.
=> 프레임 안의 객체가 프레임을 벗어나면 동등 레벨 객체가 됩니다.


4. 피그마 기능
4-1. Figma 파일에 들어간다. -> 왼쪽 상단의 파일 이름 근처의 아래 화살표를 클릭한다. -> Duplicate를 누른다.
=> 복사완료
4-2. shift 위, 아래 => 10씩 증가하거나 작아진다.
4-4. 프레임 복사하기 => 전체를 선택한 이후 alt를 사용해서 복제할 수 있습니다.
