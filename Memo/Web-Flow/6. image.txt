** 항상 디자인 할 때 이미지가 있다면 이미지와 적절한 (유사한) 서채, 색깔을 사용하거나
 주제와 관련된 서채, 색깔, 느낌을 항상 고려해야 합니다. **

1. 이미지 오버레이
=> 이미지 위에 색을 채우는 기법을 의미합니다.
=> 이미지 객체 fill 속성에서 +를 클릭하면 불투명도를 조절할 수 있습니다.
=> 투명도 조절의 가장 큰 장점은 단락을 읽기 쉬워진다는 것입니다.


2. 이미지 조절 방법
2-1. Layout -> Demensions 옆에 아이콘을 활성화시키면 비율을 유지한 채 줄어들거나 커집니다.
 아이콘을 비활성화시키면 그대로 작아지거나 커집니다.
2-2. image 수평으로 뒤집기 -> 좌클릭 -> Flip horizontal를 클릭하여 좌-우 반전할 수 있습니다.
2-3. image 수직으로 뒤집기 -> 좌클릭 -> Flip vertical를 클릭하여 상-하 반전할 수 있습니다.


3. 틴팅
=> 채도를 조절하는 방법입니다.
=> 이미지 객체의 fill 속성에서 Saturation을 조절하여 진하게 만들거나 연하게 만들 수 있습니다.


4. 익스트림 크롭
=> 사진을 확대하는 기법입니다. 작은 프레임에 이미지를 강조할 떄 주로 사용합니다.
=> 너무 확대해서 물체를 식별할 수 없을 정도는 피해야 합니다.
4-1. 사진을 프레임 안에 넣고 적당한 SIZE로 자릅니다.
4-2. 이후 사진 객체의 fill 속성에서 crop 속성을 선택하고 사진 크기를 넓힌 다음 집중할 구역을 잘라 넣습니다.
4-3. 이 떄 사진은 가장 자리에 딱 붙는게 좋습니다.


5. 소프트 크롭
=> 인위적으로 사진의 한 방향을 흐르게 처리하는 기법입니다. 세로 사진배경을 사용할 떄 주로 사용합니다.
=> 흐르게 처리하는 공간과 다른 단락 사이에 뚜렷한 경계를 피해야 합니다.
5-1. 높이를 다 채우는 직사각형 객체를 생성합니다. 넓이는 그라데이션 영역을 의미합니다.
5-2. 직사각형 객체에서 fill 속성에서 gradient 세부 속성을 클릭하고 같은 color, 다른 불투명도를 사용합니다.
5-3. 프레임은 가로가 긴 방향이기에 그라데이션 방향은 오른쪽에서 왼쪽, 왼쪽에서 오른쪽을 선택합니다. (Rotate gradient로 변경가능)
5-4. 그라데이션 처리가 완료된 객체를 이미지의 가장자리에 넣습니다. (이미지를 넘치면 이상해짐)
(완성된 그라데이션 객체를 복사하면 더 진한 그라데이션 효과를 줄 수 있습니다.)
(이미지와 관련된 색상1, 이를 편하게 나열하는 색상2를 사용하고 이미지와 관련된 색상은 이미지쪽으로 하여 배치하면 됩니다.)


6. 3등분 법칙
=> 화면을 수직 및 수평으로 3등분하면 선의 교차점은 시각적으로 
 흥미롭게 피사체를 배치할 수 있는 지점이 됩니다.
=> 그러나 시선을 확실하게 끄는 배치는 중앙 배치가 좋을 떄도 있습니다.


7. 사진 배치 +
=> 사진을 배치할 떄 무조건 사각형을 고집할 필요가 없습니다.
=> 사진에서 배경색을 제거하고 확대하는 방식
=> 사진과 콘텐츠 부분을 나뉘었지만 사진이 일부 콘텐츠 영역을 가져가는 방식..


8. 사진 선택 고려 요소
8-1. 사진에서 현실감이 느껴지는가?
8-2. 사진에서 가짜 감정이 아닌 진짜 감정이 느껴지는가?
8-3. 사진은 추상적인 것보다 구체적인것이 좋다.
