1. 시작
=> new site를 누르고 템플릿을 선택
=> 빈 페이지를 디자이너 또는 캔버스라고 부릅니다.


2. 패널
=> 왼쪽 부근의 메뉴들을 패널이라고 부릅니다.
2-1. 가장 첫 번쨰 + 버튼은 페이지에 새로운 요소를 추가하는 패널입니다.
2-2. 두 번쨰 버튼은 페이지 패널이라고 하며 웹사이트 내 모든 페이지를 보여줍니다.
2-3. 세 번쨰는 네비게이터 패널이라고 하며 피그마에서 Layer 패널과 비슷합니다.


3. 센터 메뉴에는 다양한 기기에 맞는 뷰가 존재합니다.
=> 이를 통해 반응형으로 만들 수 있습니다.


4. 오른쪽 위 Publish 버튼을 통해 만든 캔버스를 테스트할 수 있습니다.


5. 스타일 패널
=> 오른쪽에 있는 패널을 의미합니다.
=> 모든 요소를 조작하고 스타일링 합니다.


6. Layer를 통해 계층구조를 확인할 수 있습니다.
=> 부모요소 - 자식요소를 쉽게 식별할 수 있습니다.
=> 자식요소에게 상속된 CSS 속성은 오른쪽 스타일 패널에서 주황색으로 확인할 수 있습니다.
