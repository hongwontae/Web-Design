1. 패딩과 마진
=> 속성 패널의 간격 카테고리에서 조절할 수 있습니다.
1-1. 박스 모형에 패딩과 마진을 표현하고 있고 드래그하여 다방면을 조절할 수 있습니다.
1-2. alt 키를 눌러서 동방향을 같이 조절할 수 있습니다.
1-3. shift 키를 누른다면 4방향을 같이 조절할 수 있습니다.


2. 10vw
=> 요소의 각 마진에 10vw를 줘서 브라우저의 넓이 상황이 어떻게 달라지든 10%는 margin으로 처리할 수 있습니다.
=> ** 새롭게 배운 꿀팁 *


3. WebFlow Font
3-1. 폰트를 추가하려면 Site Settings에서 추가합니다.
=> 다양한 폰트의 굵기를 직접 선택해야 합니다.


4. WebFlow css class
=> 속성 패널에서 Style selector를 통해 class를 사용할 수 있습니다.


5. Image
5-1. 피그마에서 각 요소 이미지를 따로 불러와 web flow에서 설계하는 것은 불필요합니다.
5-2. 피그마에서 단체 이미지처럼 내보내고 싶다면 그룹화하여 내보내야 합니다.
5-3. 피그마에서 이미지를 내보낼 떄 Export 속성을 사용하고 몇 배의 사이즈로 내보낼 지 선택할 수 있습니다.
5-4. 또한 내보낼 떄 배경색을 제거하고 싶다면 PNG가 적합합니다.

5-5. WebFlow에서 사진을 가져올 떄는 + -> image -> 더블클릭 -> upload -> 원래 사이즈를 유지하려면 Hidpi 클릭
