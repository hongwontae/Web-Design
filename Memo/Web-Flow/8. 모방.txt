 들을 살펴보며 모방하거나 재창조하여 나의 능력을 삼아야 합니다.


2. 영감을 얻는 사이트
2-1. 드리블.com => 여러 디자인 시연
2-2. www.lapa.ninja => 홈페이지의 디자인 시연

2-4. pinterest => 여러 디자인 시연


3. 무드보드
=> 디자이너들이 영감을 얻기 위해 다양한 작업을 모아놓은 일이나 곳을 무드보드라고 합니다.


4. 모방하는 방법
4-1. 모방할 페이지를 옆에 띄어놓고 똑같은 크기의 프레임을 생성합니다. 그리고 빈 프레임이 위로 가게 모방할 페이지를 붙여넣습니다.
4-2. 레이아웃을 맞추기 위해 수직선이나 수평선이 필요합니다. 메뉴에서 Show rulers를 클릭하여 생성할 수 있습니다.
=> 수평선이나 수직선을 생성하려면 줄자를 드래그하여 가져올 수 있습니다. (이 떄 겹친 모방 페이지를 보면서 작업합니다.)
4-3. 글자 크기만큼 글씨 객체 크기를 조절하려면 객체의 속성 중 
 layout에서 resazing에서 Auto Width, Auto Height를 클릭하면 됩니다.
4-4. 1객체 안에 2객체가 있을 경우 1 객체 크기 내부에서 2객체의 정렬을 하고 싶다면 
 1,2 객체 모두 선택 후 position 속성에서 Alignment를 클릭하여 조절하면 됩니다.
4-5. 객체를 미세 이동할 떄는 1px은 화살표로 이동하고 shift+화살표는 10px 만큼 이동합니다.
