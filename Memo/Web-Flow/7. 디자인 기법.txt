** 좋은 디자인은 규칙과 기법으로 만들 수 있다. **

1. 대비
=> 개체와 또 다른 개체의 적절한 차이를 이용한 디자인 기법입니다.
=> 색상의 대비, 크기의 대비, 무게감의 대비로 나눌 수 있습니다.
=> 색상의 대비는 이미지의 색상을 가져와서 단락에 사용, 크기의 대비는 제목과 단락으로, 
 무게감의 대비는 제목 안에서 차이를 둘 수 있습니다.


2. 여백
=> 여백을 사용해서 집중할 요소를 강조할 수 있습니다.
=> 이 떄 집중할 요소는 서로 떨어져 있는 것보다 모이는 편이 좋습니다.
(이렇게 되면 여백이 많아서 불안할 수 있지만 강조할 요소를 돋보이게 하는 좋은 방법입니다.)


3. 반복
=> 색상, 사이즈, 무게감을 반복하는 디자인 기법입니다. 이를 통해 통일성을 보여줄 수 있습니다.
=> 또한 위치(레이아웃)를 통한 반복도 있습니다.


4. 일관성
=> 웹 페이지에 일정한 패턴을 지키는 기법입니다.
4-1. 레이아웃 일관성은 그리드를 사용하여 하나의 정렬 방식을 지키는 것이 좋습니다.
4-2. 서채 일관성은 목적별로 서채를 지정하여 사용하는게 좋습니다. (HeadLine은 Roboto, Subtitle은 Serif..)
4-3. 색상 일관성은 서로 관련된 색상을 사용하는 것이 좋습니다.
4-4. 이미지 일관성은 이미지의 색상, 이미지의 표정, 이미지 외모 등.. 여러 면을 고려하여 일관성을 갖추는 것이 좋습니다.


5. 겹치기
=> 서로 다른 요소들을 합치는 것을 의미합니다.
=> 제목과 사람을 겹치거나, 그림끼리 겹치거나, 색상과 겹치는 등.. 여러 방법이 존재합니다.
(사진 객체와 동일한 도형 객체를 생성하고 스포이드를 통해서 색을 가져와서 옆면에 붙이는 행위가 대표적입니다.)


6. 긴장감
=> 의도적으로 디자인 기법을 무시하고 흥미를 주는 방식입니다.
=> 일부 요소를 정렬에서 벗어나게 하는 방식, 이미지를 겹치거나 일부를 자르거나 확대하는 방식 등..
=> 디자인에서 도형을 사용하여 창의적으로 이미지를 자르는 것을 마스킹이라고 합니다.
(도형을 더블클릭하거나 패널의 객체 편집을 통해서 도형 객체의 일부 크기를 변형할 수 있습니다.)
