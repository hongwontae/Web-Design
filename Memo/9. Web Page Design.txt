1. 치수
=> 정사각형 객체를 생성하고 그 넓이나 높이를 확인하여 치수를 확인하는 것이 대표적인 방법입니다.
** 객체를 선택하고 alt를 누르고 다른 객체에 마우스를 갇다되면 거리를 알 수 있습니다. **


2. 잠금
=> 어떤 객체를 선택하거나 변경하고 싶지 않다면 왼쪽에 Layers에서 해당되는 객체를 잠금 클릭하여 선택을 막을 수 있습니다.


3 스트로크 속성
=> 객체의 선과 관련된 속성입니다.
=> 객체 색상을 배경색과 같게 하고 테두리를 대비되게 하고 싶다면 객체의 fill 색상을 지우고 stroke 속성에서
 대비되는 테두리 색을 선택하면 됩니다.


4. 그룹 만들기
=> 여러 객체를 전체 선택 후에 ctrl+z나 좌클릭 Group Selection을 누르면 선택된 객체들이 그룹이 됩니다.


5. 피그마 파일 가져오는 방법
=> Figma Community, UX Crush.. 등 다양한 곳에서 피그마 파일을 가져올 수 있습니다.
=> 이 떄 현 드래프트에서 다른 드래프트에 있는 프레임이나 객체를 가져오는 방법은
 복사-붙여넣기, 드래프트의 객체를 컴포넌트로 만든 후 Assets 탭을 클릭하여 사용할 드래프트에서 Enable하는 방식.. 등


6. 피그마 파일을 내보내는 방법
=> Export 속성을 통해 svg, png.. 등의 확장자로 내보낼 수 있습니다.


7. 그림자 효과
=> 객체의 effects 속성에서 Color, x(수평), y(수직)으로 그림자를 조절할 수 있습니다.
(이미지, 버튼, 카드.. 등 일부의 요소의 경우 수직 그림자만 적용된것이 제일 적합합니다.)

8. svg
=> 피그마에서 아이콘이나 일러스트레이션 항목은 svg가 적합합니다.
=> 백터 기반이기에 이미지를 확대하거나 축소해도 품질 저하가 없습니다.
=> 이미지가 커져도 크기가 커지지 않습니다.
=> 또한 svg는 피그마내에서 도형 취급이 되기에 자유롭게 편집할 수 있습니다.
=> flaticon보다 구글 아이콘을 사용 -> 무료이며, svg를 줍니다.


9. tidy up
=> 수직,수평 방향으로 정렬 여부를 자동으로 판단해서, 일정한 간격으로 정렬되도록 정리해줍니다.
=> 여러 객체들을 선택하고 position 속성에서 Tidy up를 클릭하면 됩니다.
=> 정렬이 되었다면 Layout의 Spacing으로 일정한 간격을 조절할 수 있습니다.
